# codecord
It's an social media app for tech enthusiasts

-It's an community web-app where you can interact with other people in the room of similar interest
-There are room created with some topics like python,100daysofCode (you can make your own room as well) where you can chat with other people and see the particiapnts of that 
room
-You can also make your own room with anytopic and you can locate it in your profile section

TECHSTACK - Django, HTML,CSS,JAVASCRIPT
